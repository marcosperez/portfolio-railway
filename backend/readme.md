# Resume

App develop with expres.js and typescript

## Modules

- Common
  - Healthcheck
  - Metrics
- Users
  - Temporary Users
  - List Users
- Videos
  - Add videos
  - List Videos with filters

### Security

- `helmet` default
- `cors` default

### Logs

- `morgan` on file and stdout

### Dependency Inyection and Inversion control

- `inversify` and `inversify-express-utils`
